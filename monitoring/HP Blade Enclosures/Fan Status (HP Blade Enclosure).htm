Overview

Service Name:

Fan Status (HP Blade Enclosure)

Service ID:

24123

Service Type:

SNMP

Supported Devices / OS:

HP Blade Enclosures

Maximum Service Instances / Device:

100

Scheduler Type:

Interval Based Scheduler

Time to Stale:

30

Service Description:

This service monitors the status of a fan in an HP blade enclosure, and provides asset data (part numbers, serial numbers, etc…) about the fan.

MIB Module(s):

CPQRACK-MIB

Notes

Configuration

You must specify a valid ‘Fan Location’. To find out the locations of the fans, query the blade enclosure with the 1.3.6.1.4.1.232.22.2.3.1.3.1.6 SNMP OID.

Errors

Misconfigured on the dashboard probably means that either the wrong SNMP community string has been specified for the device, or that the service has been configured with an incorrect Fan Location.

Comments

 The Fan Condition metric is mapped as follows:

 

Normal: 2

Warning: 4

Failed: 1,3

 

Here are the meanings of those numbers:

 

1: Other

2: OK

3: Degraded

4: Failed
