Notes

Configuration

The RAID array name needs to be specified. For a list of the RAID arrays in an EqualLogic SAN, query the SAN with the 1.3.6.1.4.1.12740.2.1.1.1.9 SNMP OID.

Errors

Misconfigured on the dashboard probably means that the SNMP community string of the device isn’t correct, or that an invalid RAID array name has been specified.

Comments

The “RAID Status” metric is actually mapped:

 

Normal (displayed as 0 in the N-central UI) = 1

Warning (displayed as 1 in the N-central UI) = 7,8

Failed (displayed as 2 in the N-central UI) = 2-6

 

1 = OK

2 = Degraded

3 = Verifying

4 = Reconstructing

5 = Failed

6 = Catastrophic Loss

7 = Expanding

8 = Mirroring
