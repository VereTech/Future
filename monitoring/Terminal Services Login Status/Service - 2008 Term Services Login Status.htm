WMI Queries Made

 
This service looks in the Win32_TerminalServiceSetting WMI class (in the root\CIMV2\TerminalServices WMI namespace) for the following properties:
 
Logons

Errors

Misconfigured on the dashboard probably means that the service has been applied to a device running the wrong version of Windows. The WMI namespace the WMI class is in differs between Windows 2003 and Windows 2008.

 

It can also mean that the Windows probe/agent failed to establish a connection to the device, either because of a network problem or because of a permissions problem.

Supported Operating Systems

This WMI query is supported on Terminal Servers and Citrix XenApp servers regardless of the level of Citrix Licensing that is installed.

If you are running Citrix XenApp Enterprise or above licensing then there is a WMI query against the root\CITRIX namespace that returns an equivalent value. The root\CITRIX namespace is not supported if you only have Citrix Advanced licensing.
