Overview

Service Name:

Windows Server Backup

Service ID:

24193

Service Type:

Windows Servers

Supported Devices / OS:

Any WMI-enabled Windows 2008 or 2008 R2 Server with the Windows Server Backup Command Line Tools installed.

Maximum Service Instances / Device:

1

Time to Stale:

30

Service Description:

Monitors the status of the backups performed by Windows Server Backup. This service requires that the ‘Windows Server Backup.PS1’ script be run on a scheduled basis (i.e. once a day) to populate WMI with updated backup information.

 

This service looks in the root\cimv2\Ncentral WMI namespace for the WindowsServerBackup_Data WMI class. It pulls the following properties:

 

NextBackupTime
