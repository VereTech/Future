Notes

Configuration 

You must specify the name of the Add-On to be monitored. For a list of installed Add-Ons, query the Lotus Foundations server with the 1.3.6.1.4.1.2.6.226.1.10.1.3 SNMP OID.

Errors

Misconfigured probably means that the SNMP community string of the device has been set incorrectly in N-central, or that an incorrect ‘Add-On Name’ has been specified.
