Overview

Service Name:

Disk I/O

Service ID:

24180

Supported Devices / OS:

Any Windows device.

Maximum Service Instances / Device:

100

Service Description:

Monitors the Total/Read/Write Operations/sec, as well as the Total/Read/Write Bytes/sec. This service looks in the root\cimv2 WMI namespace for the Win32_PerfRawData_PerfDisk_PhysicalDisk WMI class. It pulls the following properties:

 

DiskWritesPerSec

DiskReadsPerSec

DiskBytesPerSec

DiskReadBytesPerSec

DiskWriteBytesPerSec

Timestamp_Sys100NS

Frequency_Sys100NS

Notes

Configuration 

You must specify the name of the physical disk to be monitored. For a list of the physical disks on a specific device, look at Name property of each instance in the Win32_PerfRawData_PerfDisk_PhysicalDisk WMI class, under the root\cimv2 WMI namespace.

Errors

Misconfigured on the dashboard probably means that an invalid physical disk name has been specified, or that a WMI communication issue is occurring.
