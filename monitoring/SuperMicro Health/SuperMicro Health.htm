Service Description:

Monitors the output reading of a hardware sensor on a SuperMicro server, as well as whether or not the sensor is enabled. The SuperDoctor III application can be downloaded from the following link:

 

http://www.supermicro.com/products/accessories/software/superodoctoriii.cfm

Notes

Configuration

To configure this service, you will have to specify the name of the sensor to be monitored. For a list of sensors, query the SuperMicro server with the 1.3.6.1.4.1.10876.2.1.1.1.1.2 SNMP OID.

 

As each type of sensor (fan speed, for example, versus power supply readings) will have very different thresholds, you’ll need to customize the thresholds for the Sensor Reading metric on a per-sensor basis.

Errors

Misconfigured on the dashboard probably means that either the wrong SNMP community string has been specified for the device, or that the service has been configured with a non-existent sensor name.
