Configuration

Comments

The Phase Status metric is mapped. Here are the values that are available through SNMP:

 

1: Other

2: Unknown

3: OK

4: Non Critical

5: Critical

6: Non Recoverable

 

This service maps those values as follows:

 

Normal: 3

Warning: 1, 2, 4

Failed: 5, 6
