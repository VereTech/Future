Errors

Misconfigured probably means that the SNMP community string of the device has been set incorrectly in N-central.

Comments

The Battery Status metric returns the following values:

 

1 = The Battery is Charging

2 = The Battery is Discharging

3 = The Battery is being Charged to Its Float Voltage

4 = The Battery is Resting

5 = Unknown

 

Those states are mapped as follows:

Normal = 1, 3, 4

Failed = 2, 5
