Time to Stale:

Service Long Description:

Monitors the remaining battery run-time and charge level, as well as any line-in errors. Also monitors whether or not the UPS is ‘on-battery’, and if there are any environmental issues (for example, extreme temperature or humidity readings). monitors the Capacity Load on the UPS, and it's reported

Notes

Errors

Misconfigured probably means that the SNMP community string of the device has been set incorrectly in N-central, or that an incorrect SNMP index has been specified.

 

To determine what SNMP indices are valid, query the UPS with the 1.3.6.1.4.1.705.1.5.1 SNMP OID.
