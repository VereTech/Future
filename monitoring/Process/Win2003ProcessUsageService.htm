Notes

Configuration 

You must specify the name of the name of the process that you would like to be monitored. For a list of the message process on a specific device, add in the Windows Process Service and then go to the Service Details Tab followed by selecting the select Discovered drop down menu bar.  For example if  the process name returned from the process service is agent.exe, simply put in agent as the process name.
You should also adjust the thresholds of the memory usage service to match the maximum amount of RAM you want allocated for the process being monitored

Errors

Misconfigured on the dashboard probably means that an invalid process name has been specified, or that a WMI communication issue is occurring.
